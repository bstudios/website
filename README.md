Initial Setup &amp; Theme based on the incredible site by [York Community Web and App Development Society](https://github.com/yordevs/yordevs.github.io)

## Local Setup

```sh
yarn install
gatsby develop
```

## Common Issues

### Gatsby CLI not installed

```sh
npm install -g gatsby-cli
```

### Yarn not installed

```sh
npm install -g yarn
```
